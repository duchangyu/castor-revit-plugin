## 版本历史说明

- **Major Version (主版本)**: 不兼容的 API 修改
- **Minor Version (次版本)**: 向后兼容的功能新增
- **Patch Version (补丁版本)**: 向后兼容的错误修复

### 版本标签格式

- `feat`: 新功能
- `fix`: 错误修复
- `docs`: 文档更新
- `refactor`: 代码重构
- `perf`: 性能优化
- `test`: 测试相关
- `chore**: 构建/工具相关
